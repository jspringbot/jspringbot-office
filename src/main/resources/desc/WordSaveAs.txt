This will save as the opened document. Also allows to convert to DOC or DOCX Microsoft Word.

| Word Save As | /Users/user1/Desktop/a.doc |
| Word Save As | /Users/user1/Desktop/a.docx | DOCX |
| Word Save As | /Users/user1/Desktop/a.doc | DOC |

=>

| This save the document to "a.doc". This should retain the same document format.
| This save the document to "a.docx". This converts the opened file to DOCX format.
| This save the document to "a.doc". This converts the opened file to DOC format.
